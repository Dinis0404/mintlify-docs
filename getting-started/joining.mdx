Joining DirtShop SMP is quick and straightforward — all you need is a copy of Minecraft: Java Edition and a few seconds to add the server to your multiplayer list. This guide walks you through the entire process, from opening Minecraft to spawning in for the very first time.

<Note>
  DirtShop SMP is a **Java Edition** server only. Bedrock Edition (Windows 10/11 app, console, or mobile) is not supported. Make sure you're launching the Java Edition version of Minecraft before attempting to connect.
</Note>

## Server Details

| Detail | Value |
|---|---|
| **Server IP** | `play.dirtshop.gg` |
| **Edition** | Java Edition |
| **Supported Versions** | Latest stable release and one version back |

The server keeps up with current Minecraft releases. You'll have the best experience on the latest stable Java Edition version, though the previous major version is also fully supported if you haven't updated yet.

## Adding the Server to Minecraft

<Steps>
  <Step title="Open Minecraft: Java Edition">
    Launch the Minecraft Launcher, make sure the **Java Edition** profile is selected, and click **Play**.
  </Step>
  <Step title="Navigate to Multiplayer">
    From the main menu, click **Multiplayer**. If a warning appears about online play, accept it to continue.
  </Step>
  <Step title="Add a new server">
    Click **Add Server** in the bottom-left area of the multiplayer screen.
  </Step>
  <Step title="Enter the server details">
    In the **Server Name** field, type something memorable like `DirtShop SMP`. In the **Server Address** field, type:

    ```
    play.dirtshop.gg
    ```

    Leave the port as default — no port number is needed.
  </Step>
  <Step title="Save and connect">
    Click **Done** to save the entry. DirtShop SMP will now appear in your server list. Click on it once to let it ping and confirm it's online, then click **Join Server** to connect.
  </Step>
</Steps>

## What to Expect on Your First Login

When you connect for the first time, you'll be placed in the **spawn area** — a central hub that orients new players and provides access to key parts of the server. Here's what happens automatically:

- **Welcome message** — A greeting message is displayed in chat with a quick summary of helpful commands and a link to the rules.
- **Starter kit** — New players receive a one-time starter kit containing basic tools and supplies to help you get on your feet. Check your inventory after joining.
- **Spawn hub access** — The spawn area contains information boards, warps to the player market, and signposts for key locations. Take a moment to look around before heading out.

When you're ready to leave spawn and start exploring, run `/rtp` to teleport to a random location in the wild, or ask in chat — other players are usually happy to point you in the right direction.

<Tip>
  If you're having trouble connecting — whether the server appears offline, you get a version mismatch error, or you can't get past the login screen — head to the [DirtShop SMP Discord](/community/discord). The `#support` channel is the fastest way to get help from staff and the community.
</Tip>
