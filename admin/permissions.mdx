DirtShop SMP uses a tiered rank system managed by LuckPerms. As you invest more time in the server, you automatically unlock new commands and quality-of-life features. Each rank inherits all permissions from the rank below it, so nothing you've earned is ever taken away as you progress. Use `/rankup info` at any time to see your current rank, your total playtime, and exactly how much more time is needed to reach the next tier. This page is the definitive reference for what each rank can do.

## Rank Permissions

<Accordion title="Newcomer">
  Newcomer is the starting rank granted to every player on their first join. These commands cover the basics of navigating the server.

  - `/spawn` — teleport to the main spawn
  - `/help` — list available commands
  - `/rules` — read the server rules
  - `/msg <player> <message>` — send a private message
  - `/r <message>` — reply to the last private message
  - `/afk` — mark yourself as away from keyboard
  - `/list` — see all online players
  - `/seen <player>` — check when a player was last online
  - `/rtp` — randomly teleport into the wild
</Accordion>

<Accordion title="Member (+ Newcomer)">
  Member rank unlocks teleportation tools and the ability to set your first homes. Earned after completing the in-game quiz with `/quiz`.

  - `/tpa <player>` — request to teleport to another player
  - `/tpahere <player>` — request another player teleport to you
  - `/tpaccept` — accept an incoming teleport request
  - `/tpdeny` — deny an incoming teleport request
  - `/sethome <name>` — set up to **3 homes**
  - `/home <name>` — teleport to a saved home
  - `/delhome <name>` — delete a saved home
  - `/quiz` — complete the server rules quiz
</Accordion>

<Accordion title="Regular (+ Member)">
  Regular rank is earned after 24 hours of playtime. It unlocks jobs, voting, and two additional home slots.

  - `/sethome <name>` — now supports up to **5 homes**
  - `/jobs` — access the Jobs Reborn menu
  - `/vote` — get your voting links to earn coin rewards
  - `/playtime` — check your total time on the server
  - `/ignore <player>` — block chat messages from a specific player
</Accordion>

<Accordion title="Veteran (+ Regular)">
  Veteran rank rewards long-term commitment with personal expression features and a significant home expansion.

  - `/sethome <name>` — now supports up to **10 homes**
  - **Colored chat** — use `&` color codes in chat messages
  - `/nick <nickname>` — set a custom display name (your own name only; impersonation is not allowed)
</Accordion>

<Accordion title="Elite (+ Veteran)">
  Elite rank introduces one of the most requested features: flight — with a reasonable limitation to keep survival balanced.

  - `/sethome <name>` — now supports up to **15 homes**
  - `/fly` — fly freely, but only within your own claimed land
  - `/fix hand` — repair the item in your main hand using experience levels
</Accordion>

<Accordion title="Legend (+ Elite)">
  Legend is the highest playtime rank on DirtShop SMP, reserved for the most dedicated long-term players.

  - `/sethome <name>` — now supports up to **20 homes**
  - **Custom join message** — configure a personalized message that displays when you join the server
  - **Particle effects** — toggle cosmetic particle effects around your character
  - **Priority queue** — skip ahead in the server join queue when the server is full
</Accordion>

<Accordion title="Donor Ranks (VIP / VIP+ / MVP)">
  Donor ranks are purchased separately and run alongside your playtime rank — they don't replace it. VIP, VIP+, and MVP grant cosmetic perks and convenience features as a thank-you for supporting the server financially. For the full breakdown of what each tier includes, see the **Rank Perks** page.
</Accordion>

<Note>
  Staff ranks (Helper, Moderator, Senior Moderator, and Admin) carry additional permissions not listed on this page. These include moderation commands, log inspection tools, and administrative utilities that are granted on a need-to-have basis and are not available to regular players.
</Note>
