Every feature on DirtShop SMP — from land claiming and player shops to the live world map — is powered by a carefully selected set of plugins. This page gives you a plain-language overview of what's running under the hood, so you understand where your commands come from and who to ask when something doesn't work as expected. You don't need to know anything technical to benefit from this list.

## Installed Plugins

| Plugin | What It Does |
|--------|-------------|
| EssentialsX | Core commands: homes, warps, teleport, economy |
| GriefPrevention | Land claiming with a golden shovel |
| Vault | Economy framework connecting all shop and money plugins |
| ChestShop | Lets players create buy/sell chest shops |
| LuckPerms | Manages rank permissions and chat prefixes |
| Jobs Reborn | Earn coins for mining, farming, fishing, and more |
| Dynmap | Live browser-based map of the server at map.dirtshop.gg |
| CoreProtect | Logs all block changes for grief rollback |
| Votifier | Rewards players with coins for voting on server lists |
| PlaceholderAPI | Powers custom scoreboards and tab lists |
| WorldBorder | Enforces the 20,000-block world border |

## Plugin Highlights

<CardGroup cols={2}>
  <Card title="GriefPrevention" icon="shield">
    Protect your builds by claiming land with a golden shovel. Claims block other players from breaking, placing, or accessing containers in your area. Run `/claiminfo` while standing in a claim to see who owns it.
  </Card>
  <Card title="Jobs Reborn" icon="briefcase">
    Earn coins passively by doing what you already love — mine ores, fish, farm crops, hunt mobs, and more. Join up to two jobs with `/jobs join` and track your progress with `/jobs stats`.
  </Card>
  <Card title="ChestShop" icon="store">
    Set up a player-run shop by placing a chest, a sign, and writing the right format. Other players can buy from or sell to your shop automatically, even when you're offline.
  </Card>
  <Card title="CoreProtect" icon="clock-rotate-left">
    Every block change on the server is logged. If your build gets griefed, staff can inspect the damage with `/co inspect` and roll it back completely. No grief is permanent.
  </Card>
</CardGroup>

<Tip>
  Explore the entire server world from your browser using the live Dynmap at [map.dirtshop.gg](https://map.dirtshop.gg). You can see player locations, biomes, claims, and points of interest in real time — no login required.
</Tip>

<Note>
  The plugin list changes as the server updates and improves. Plugins may be added, removed, or replaced without notice during version upgrades. Always check **#announcements** in the Discord for details on plugin changes and what they mean for your gameplay.
</Note>
