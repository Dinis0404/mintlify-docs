DirtShop SMP runs with a specific set of world settings and game rules designed to keep survival challenging without making the experience unfair or frustrating. This page documents the server's current configuration so you know exactly what to expect before diving in — from difficulty and world size to how TNT and mob griefing are handled.

## Server and World Configuration

| Setting | Value |
|---------|-------|
| Server Version | Java Edition 1.21.x |
| Difficulty | Hard |
| Game Mode | Survival |
| PvP | Enabled (except spawn and protected zones) |
| World Border | 20,000 × 20,000 blocks |
| Overworld Seed | Hidden |
| Nether | Enabled (1:8 scale) |
| The End | Enabled |
| Resource World | Resets monthly |
| Keep Inventory | Off (default) |
| Fire Spread | Off |
| TNT Damage | Off in claims; On in the wild |
| Mob Griefing | Off (creeper explosions don't damage terrain) |
| Weather | Enabled |

## PvP Zones

PvP is enabled by default anywhere in the wild — outside of spawn and protected areas, you engage other players at your own risk. If you prefer to play cooperatively on your own land, you can disable PvP in your own claim at any time using `/pvp toggle`. This setting only affects your claimed territory and does not carry over to other players' claims.

The spawn area is always a PvP-free zone. No matter your settings or rank, combat is completely disabled within the spawn region. Use spawn as a safe meeting point or trade hub without worrying about being attacked.

## Claim Settings

Every player starts with **500 claim blocks**, which is enough to protect a solid starter base. You earn additional claim blocks automatically the longer you play: **100 blocks per hour** of active playtime. There is no cap on how many claim blocks you can accumulate over time.

To check your current claim block balance, use `/claimslist`. If you need more blocks quickly, claim blocks can also be purchased in the server shop.

## Spawn Protection

A **100-block radius** centered on coordinates 0, 0 is protected by server-level admin protection. No player, regardless of rank, can place or break blocks in this zone through normal gameplay. Building in the spawn area is only permitted in designated player zones explicitly set up by an Admin.

If you accidentally wander into the spawn protection radius and find commands blocked, simply move further out into the world or use `/rtp` to teleport to the wild.

<Note>
  Game rule changes — including adjustments to difficulty, PvP settings, or world borders — are announced in **#announcements** on Discord before they take effect. If something about the server feels different, check the announcements channel first.
</Note>
