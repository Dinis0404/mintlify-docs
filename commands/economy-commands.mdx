DirtShop SMP runs on a player-driven coin economy — you earn coins through jobs, selling items, and trading with other players, then spend them at the server shop, player market, and beyond. These economy commands give you full control over your finances, from checking your balance and paying friends to browsing shops and climbing the wealth leaderboard.

## Balance & Payments

Keep track of your coins and send payments to other players using these core balance commands.

| Command | Description |
|---------|-------------|
| `/money` | Check your current coin balance |
| `/bal` | Alias for `/money` |
| `/pay <player> <amount>` | Send coins to another player |
| `/baltop` | View the wealthiest players |

## Shop Commands

Whether you're buying from the server shop or selling off a full inventory haul, these commands make trading quick and straightforward.

| Command | Description |
|---------|-------------|
| `/shop` | Open the server shop GUI |
| `/shops` | Browse active player shops |
| `/sellall` | Sell all items in your inventory to the server shop |
| `/sell hand` | Sell the item you're holding |

<Info>
  Not all items are accepted by the server shop. If `/sellall` or `/sell hand` doesn't register an item, check `/warp market` to see if another player is buying it instead.
</Info>

## Jobs Commands

Jobs let you earn coins automatically by doing what you already love — mining, farming, building, fishing, and more. Each job has its own pay rates for different activities, so you can pick the ones that suit your playstyle.

| Command | Description |
|---------|-------------|
| `/jobs browse` | View available jobs |
| `/jobs join <job>` | Join a job to earn coins for activities |
| `/jobs leave <job>` | Leave a job |
| `/jobs info <job>` | View a job's pay rates |

<Tip>
  Jobs are one of the best ways to build your wealth passively. Once you join a job, you earn coins simply by playing — no extra steps required. Run `/jobs browse` and pick a job that matches what you already do on the server, then let the coins roll in while you play normally.
</Tip>

## Quick Reference

Use this snippet as a handy cheat sheet for the most common economy commands:

```bash
/money             # Check balance
/pay Steve 500     # Send 500 coins to Steve
/sellall           # Sell everything in inventory
/baltop            # Leaderboard
```
