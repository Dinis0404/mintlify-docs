Getting around DirtShop SMP is fast and flexible thanks to a full suite of teleportation commands. You can bookmark your favourite builds with named homes, jump to community destinations using public warps, or send teleport requests directly to other players. The sections below cover every command you need to move around the server efficiently.

## Home Commands

Homes let you save specific locations in the world so you can return to them instantly at any time. Give each home a descriptive name so you always know where you're headed.

| Command | Description |
|---------|-------------|
| `/sethome <name>` | Save your current location as a named home |
| `/home <name>` | Teleport to a saved home |
| `/delhome <name>` | Delete a saved home |
| `/homes` | List all your saved homes |

<Info>
  The number of homes you can set depends on your rank — **Newcomer** players start with **1 home**, and this limit increases all the way up to **20 homes** at the **Legend** rank. Check the full breakdown on the [Rank Perks](/ranks/perks) page.
</Info>

## Warp Commands

Public warps are server-maintained teleport points that connect you to key locations across the server. Anyone can use them at any time, no rank requirement needed.

| Command | Description |
|---------|-------------|
| `/warp <name>` | Teleport to a public warp |
| `/warps` | List all available warps |

## Player Teleportation

You can send and receive teleport requests to travel directly to another player's location. All requests must be accepted before any teleportation occurs.

| Command | Description |
|---------|-------------|
| `/tpa <player>` | Request to teleport to a player |
| `/tpahere <player>` | Request a player to teleport to you |
| `/tpaccept` | Accept an incoming teleport request |
| `/tpdeny` | Deny a teleport request |
| `/tptoggle` | Toggle all teleport requests on/off |

<Note>
  `/tpa` and `/tpahere` require **Member rank or higher**. If you haven't taken the rules quiz yet, run `/quiz` to earn your Member rank and unlock player teleportation.
</Note>

<Warning>
  All teleportation has a **3-second warmup** before it activates. If you move, take damage, or interact with anything during this countdown, the teleport will be cancelled and you'll need to send the request again.
</Warning>
