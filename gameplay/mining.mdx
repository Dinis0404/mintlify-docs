在泥土商店中，你可以找到大大小小的礦脈與單個礦物。常見礦物包含煤炭礦、銅礦、鐵礦、金礦與綠寶石礦。

每一種礦物都會按照稀有程度定價。礦物越稀有，初始定價越高，因此同一類礦物也可能出現不同價格。

## 礦物價格

| 礦石名稱 | 價格範圍 |
| --- | --- |
| 煤炭礦 | 1 - 10 |
| 銅礦 | 1 - 20 |
| 鐵礦 | 1 - 40 |
| 金礦 | 1 - 100 |
| 綠寶石礦 | 1 - 300 |

## 礦物冷卻時間

礦物挖掘後會進入冷卻時間，冷卻中的礦物會變成基岩方塊，時間到了之後就可以再次挖掘。

| 礦石名稱 | 冷卻時間 (秒) |
| --- | ---: |
| 煤炭礦 | 20 |
| 銅礦 | 40 |
| 鐵礦 | 60 |
| 金礦 | 180 |
| 綠寶石礦 | 300 |

## 神秘礦石

挖礦除了會掉落礦物外，也有機率掉落「神秘礦石」。你可以前往 [重生市](/world/respawn-city) 找礦工，花費 `1000` 塊打開神秘礦石，換取特殊物品。

![在重生市的礦工](/images/migrated/economy-mining-miner.png)

_在重生市的礦工。_

### 不同礦物出現神秘礦石的機率

| 礦石名稱 | 神秘礦石機率 |
| --- | ---: |
| 煤炭礦 | 0% |
| 銅礦 | 1% |
| 鐵礦 | 1% |
| 金礦 | 3% |
| 綠寶石礦 | 7% |

### 神秘礦石能獲得的東西

| 物品名稱 | 機率 | 描述 |
| --- | ---: | --- |
| 細緻、粗製、優質魚餌 | 40% | 釣魚會用到的魚餌，和商店賣的一樣 |
| 相當好的鑽石鏟 | 20% | 和商店一樣，附有 效率 3 的鑽石鏟 |
| 特級釣魚竿 | 20% | 和商店一樣，附有 耐久 5、魚餌 5 的釣魚竿 |
| 金蘋果 5 個 | 40% | 普通的金蘋果 |
| 飛行時間 2 或 3 的煙火 5 個 | 5% | 普通的煙火 |
| 豬布林頭顱 | 5% | 普通的豬布林頭顱 |
| 挖更遠的鎬子 | 5% | 在主手使用時增加 `10.0` 的方塊互動距離 |
| 鞘翅 | 1% | 普通的鞘翅 |
| 礦工藥水 | 1% | 飲用後獲得速度 3、挖掘加速 3，持續 `50000` 秒 |
| 跳躍靴 | 1% | 裝備時增加 `1.0` 的跳躍力 |
| 放大皮衣 | 1% | 裝備時增加 `1.0` 的大小 |
| 釣竿 | 1% | 附有 魚餌 5、耐久 15 的釣竿 |
| 釣竿 | 1% | 附有 魚餌 7、耐久 3 的半耐久釣竿 |
| 獄髓鏟 | 1% | 附有 效率 3 的獄髓鏟 |
| (已移除) 山羊角 | / | 普通的山羊角 |
