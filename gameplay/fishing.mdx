在泥土商店中，很多地區都有人在釣魚，而且不同地區會有各自的特色魚種。魚的價格通常取決於大小與稀有程度。

## 釣竿價格

| 魚竿名稱 | 價格 | 魚竿的附魔屬性 |
| --- | ---: | --- |
| 釣魚竿 | 64 | 普通的釣魚竿 |
| 上級釣魚竿 | 350 | 耐久 3、魚餌 1 的釣魚竿 |
| 高級釣魚竿 | 1000 | 耐久 5、魚餌 3 的釣魚竿 |
| 特級釣魚竿 | 2500 | 耐久 5、魚餌 5 的釣魚竿 |

## 巨大化

釣魚途中有一定機率，約為 `1/400`，會出現「巨大化」。巨大化的魚通常體型更大，價格也更高，但同時也相當吃運氣。

![有玩家釣上巨大化時，全服通知的信息](/images/migrated/economy-fishing-giant-catch.png)

_有玩家釣上巨大化時，全服通知的信息，會顯示魚的種類、大小與價錢。_

## 魚餌系統

泥土商店提供三種不同的魚餌，用來提升釣到巨大化或稀有魚種的機率權重。使用時需要把魚餌放在背包左上角，釣魚時魚餌會自動消耗。

### 魚餌價格

| 魚餌名稱 | 價格 (1 包) | 價格 (5 包) | 用途說明 |
| --- | ---: | ---: | --- |
| 細緻魚餌 | 5 | 25 | 提升稀有魚種上鉤機率 |
| 粗製魚餌 | 10 | 50 | 提升巨大化魚種上鉤機率 |
| 優質魚餌 | 10 | 50 | 提升稀有魚種上鉤機率，但降低巨大化魚種上鉤機率 |

### 權重怎麼看？

假設某個地區有四種魚:

- A 魚，機率權重 `40`
- B 魚，機率權重 `20`
- C 魚，機率權重 `5`
- D 魚，機率權重 `1`

那麼 D 魚的出現機率就是 `1/66`。魚餌會直接提升全部魚種的機率權重，因此原本稀有的魚也會更容易釣到。

### 魚餌效果

| 魚餌名稱 | 魚餌對於釣魚中的影響 |
| --- | --- |
| 細緻魚餌 | 權重 `+10` |
| 粗製魚餌 | 當沒釣到巨大化時，有 `1/150` 的機率變成巨大化 |
| 優質魚餌 | 權重 `+35`，但釣到巨大化時有 `1/15` 的機率取消巨大化 |

## 寶藏

除了巨大化之外，釣魚途中也有一定機率，約為 `1/200`，出現寶藏。寶藏可能包含經驗、金錢，或特殊物品與裝備。

### 寶藏中能獲得的東西

| 物品名稱 | 機率 | 描述 |
| --- | ---: | --- |
| 經驗 20 XP | 50% | 普通的經驗 |
| 金錢 100 | 30% | 普通的金錢 |
| 金錢 300 | 15% | 普通的金錢 |
| 金錢 1000 | 2% | 普通的金錢 |
| 時鐘 | 1% | 普通的時鐘 |
| 望遠鏡 | 1% | 普通的望遠鏡 |
| 疾走鞋 | 1% | 裝備時增加玩家 `0.1` 的速度 |
